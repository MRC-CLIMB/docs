# **Pricing**

---

!!! info
    Please note that **our Prices will update as of 1st April 2026**. Any quotations offered and processed before this date will reflect current prices. Both current and updated prices can be found within the pages below.

CLIMB provides access through justified packages with associated prices, aligned with specific use cases:

+ [**Research package**](2.1.Research-package.md)

    The research package provides support for funded research projects, multi-institutional collaborative research, PhD and postdoctoral research activities, international research collaboration.

+ [**Training package**](2.2.Training-package.md)

    The training package provides support for bioinformatics skills development, capacity building initiatives, workshops and tutorials.

+ [**Surveillance package**](2.3.Surveillance-package.md)

    The surveillance package provides support for public health surveillance activities, pathogen monitoring and analysis, emergency response capabilities, collaboration with public health agencies.

+ [**Trial package**](2.4.Trial-package.md)

    The trial package provides support for evaluation purposes or proof-of-concept activities, transition pathway to sustained access, new user onboarding and assessment.

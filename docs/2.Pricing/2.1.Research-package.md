# **Research package plans**

---

The research package provides support for funded research projects, multi-institutional collaborative research, PhD and postdoctoral research activities, international research collaboration.

## **Starter pack**
For a single user or a very small team (not a team of bioinformaticians) we suggest:

+ 8vCPUs, 64GB RAM (JupyterLab environments/nextflow)
+ 500GB Team Shared Storage
+ 1TB S3 storage

| Resource | Price until 31st March 2026 | Price from 1st April 2026 |
|---| ----------- | ---- |
| Starter pack | starts at **£1,200/year** or **£160/month** excl. VAT | starts at **£1,300/year** or **£170/month** excl. VAT |

!!! info
    Currently, each individual user can have one JupyterLab environment (2-8vCPUs) with multiple environments possible within a team depending on the size selected. A teams quota can be increased to allow more users within a team to create larger environments at the same time.

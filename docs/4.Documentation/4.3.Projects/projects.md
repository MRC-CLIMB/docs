# **What are CLIMB Projects ?**

---

By default all teams and users fall under the CLIMB Project and are managed under CLIMBs User Support. Separate Projects and **superadmin access** is available through collaborative agreements.

Projects allow for a large number of teams and users to be managed by superadmins within an organisation. This allows for flexibility in how users and resources are managed without going directly through CLIMB support channels. Support from CLIMB is still maintained but at a higher level through infrastructure queries.

Please [**contact us**](mailto:climb@quadram.ac.uk) if you would like to discuss Project access and management for your organisation.

# **Documentation**

---

Welcome to the [**CLIMB**](https://climb.ac.uk/) documentation page ! These docs are designed to help you get the best out of the CLIMB infrastructure. But before you start, there are 2 interfaces you will be using:

+ [**Bryn**](https://bryn.climb.ac.uk) is the web interface for CLIMB, and is the primary way to access the infrastructure.
From Bryn, you can manage your team, it's resources, the S3 buckets and your notebooks. 

+ **JupyterLab environments**, on the other hand, are JupyterLab environments fully integrated with CLIMB and it's storage, with first-class support to Nextflow pipelines. Within JupyterLab environments you will be using the JupyterLab interface.
